# 🐦 GrowTwitter

> Plataforma para crescimento e engajamento no Twitter de forma inteligente 🚀

---

## 💻 Sobre o projeto

O **GrowTwitter** é uma aplicação desenvolvida para ajudar usuários a crescerem no Twitter com mais estratégia, organização e consistência.

---

## 🚀 Tecnologias utilizadas

* ⚛️ React
* ⚡ Vite
* 💙 TypeScript
* 🎨 CSS

---

## ⚙️ Como rodar o projeto

```bash
# Clonar repositório
git clone https://github.com/KarolSouzaS/Projeto.Growtwitter.git

# Entrar na pasta
cd Projeto.Growtwitter

# Instalar dependências
npm install

# Rodar projeto
npm run dev
```
